# Local Guide Connector
## Deskripsi
Menghubungkan wisatawan dengan tour guide lokal terverifikasi secara praktis dan
aman. Fokus pada SDG 8 (Pekerjaan Layak & Pertumbuhan Ekonomi).

## Fitur
- Verifikasi Guide Lokal
- Pencarian & Filter Lokasi
- Booking System

## Instalasi
### Prasyarat
- Flutter SDK
- Node.js & Nest CLI
- PostgreSQL Instance

### Running App
1. Clone repository:
`git clone [link-repo-anda]`
2. Setup Backend:
`cd backend && npm install && npm run start:dev`
3. Setup Frontend:
`cd frontend && flutter pub get && flutter run`
